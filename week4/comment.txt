1. with 문 사용: 파일 작업 시 자동으로 리소스를 관리하고 예외가 발생해도 파일이 적절히 닫히도록 컨텍스트 관리자(with 문)를 사용

2. a' 모드로 파일 열기: 로그 파일을 추가 모드('a')로 열어 이전 기록

3. 문자열 포매팅(f-strings): Python 3.6 이상에서 지원하는 f-strings를 사용하여 로그 메시지와 출력 메시지

4. random.uniform() 사용: 각 센서 값에 대해 정확한 범위 내에서 부동소수점 난수를 생성하기 위해 사용했습니다. random.randint() 대신 이것을 선택한 이유는 연속적인 값(온도, 습도 등)을 더 현실적으로 시뮬레이션하기 위함입니다.

5. datetime 모듈 사용: 로그 파일에 정확한 타임스탬프를 기록하기 위해 사용했습니다. 이는 데이터 분석과 문제 해결 시 중요한 시간 정보를 제공합니다.
