1.나눔
 DummySensor 클래스, 보조 함수(print_env_values),main

2. 상세한 문서화
 가이드라인에 맞춰 독스트링 형식을 사용

3. 객체 지향적 
 DummySensor 클래스를 통해 환경 데이터 관리를 캡슐화
 초기화(__init__), 데이터 설정(set_env), 데이터 반환 및 로깅(get_env) 기능을 메소드로 분리
 객체의 상태(환경 값)는 env_values 사전에 저장되며, 메소드를 통해서만 접근하도록 설계

4. 오류처리
 로그 파일 작성 시 예외 처리를 통해 프로그램이 예기치 않게 중단되는 것을 방지
 오류시 메시지 출력

5. 데이터 포맷팅과 출력
 print_env_values 함수이용
 구분서 사용
 단위사용

6. 모듈 실행 제어
 if __name__ == '__main__': 구문을 사용하여 모듈이 직접 실행될 때와 임포트될 때의 동작

7. 정밀한 데이터 처리
ound() 함수를 사용하여 각 센서 값이 적절한 소수점 자리까지만 표시

8. 로깅 기능
 환경 데이터를 날짜 및 시간 정보와 함께 텍스트 파일에 기록
 이를 통해 환경 값의 변화를 시간에 따라 추적할 수 있게 설계계
